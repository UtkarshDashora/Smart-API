# 🚀 Smart API Generator  
**No-Code API Builder using React & Supabase**

---

## 🌐 Overview

Smart API Generator is a powerful no-code platform that allows users to visually design database schemas and automatically generate fully functional REST APIs.

It simplifies backend development by combining schema design, API generation, authentication, and testing into a single intuitive interface.

---

## ✨ Features

- 🖱️ Visual Schema Builder (drag & drop interface)  
- 🔁 Auto-generated CRUD APIs (Create, Read, Update, Delete)  
- 🔐 Authentication (Email/Password + OAuth via Supabase)  
- 📄 Export Options:
  - OpenAPI (Swagger) documentation  
  - Supabase SQL schema  
  - Express.js backend boilerplate  
- 🔧 Live API Testing & Documentation Viewer  
- 💾 Cloud Sync with Supabase  
- 🔒 RBAC & Table Permission Controls  
- 🌐 Built-in CORS configuration  

---
